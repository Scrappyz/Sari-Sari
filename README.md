# Sari-Sari
A point of sale system for a sari-sari store.

## Tech Stack
### Frontend
- React 19

### Backend
- Java 23
- Spring Boot 3.4.3
- Maven

### Database
- PostgreSQL 17

## Usage
1. Navigate to the `backend` folder and run `mvn spring-boot:run`
2. Navigate to the `frontend` folder and run `npm run dev`
